成功提示：ShokaX 是一个精美的 Hexo 主题，本文将详细介绍如何使用 ShokaX 主题来搭建个人博客。

# 🚀 主题简介

ShokaX 是 shoka 的精神续作，致力于提高性能和优化魔改体验。它拥有以下特点：

  • PWA 支持
  • 注入 API
  • 社区插件系统
  • 自定义字体和样式
  • 多种评论系统支持
  • 底部备案号和自定义页尾
  • CSS 渐变封面
  • TypeScript 支持
