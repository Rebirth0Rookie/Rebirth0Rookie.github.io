首页教程Hexo主题ShokaX

成功提示：ShokaX 是一个精美的 Hexo 主题，本文将详细介绍如何使用 ShokaX 主题来搭建个人博客。

🚀 主题简介

ShokaX 是 shoka 的精神续作，致力于提高性能和优化魔改体验。它拥有以下特点：

  • PWA支持
  • 注入API
  • 社区插件系统
  • 自定义字体和样式
  • 多种评论系统支持
  • 底部备案号和自定义页尾
  • CSS渐变封面
  • TypeScript支持

📦 安装方法

环境准备

ShokaX 主题需要 Node.js 18.x 或更高版本才能运行，请先确保你的环境满足要求。

安装步骤

  1. 首先初始化一个 Hexo 博客：
npm install -g hexo-cli
hexo init my-blog
cd my-blog
  1. 安装 ShokaX 主题及其依赖：
npm install hexo-theme-shokax@latest
npm install hexo-renderer-multi-next-markdown-it@latest
  1. 修改 Hexo 配置文件 _config.yml，设置主题：
theme: shokax
  1. 创建主题配置文件 _config.shokax.yml，配置主题。

⚙️ 主题配置

ShokaX 的配置分为站点配置和主题配置两部分。站点配置在 _config.yml 文件中，主题配置在 _config.shokax.yml 文件中。

基本配置

# 基本设置
alternate: 你的博客名称 # 浏览器标签标题

# 头像设置
avatar:
  url: /images/avatar.jpg # 头像地址
  status:
    enable: true
    emoji: 😊
    on_mouse_over: "欢迎光临"
    on_tap: "你好啊！"

主题颜色

theme:
  dark: false # 是否默认暗色主题
  profile:
    color: '#3e80f5' # 默认主题色
    background: linear-gradient(to right, #3e80f5 0%, #a3c5ff 100%) # 渐变背景

导航菜单

menu:
  home: / || home
  文章:
    归档: /archives/ || archive
    分类: /categories/ || folder
    标签: /tags/ || tag
  友链: /friends/ || link
  关于: /about/ || user

📝 文章写作

ShokaX 支持丰富的文章格式和样式，下面是一些常用的写作技巧：

Front-matter 设置

在文章开头使用 front-matter 来设置文章的元数据：

---
title: 文章标题
date: 创建日期
updated: 更新日期
categories:
  - [一级分类, 二级分类]
tags:
  - 标签1
  - 标签2
keywords: '关键词1,关键词2'
description: 文章描述
sticky: 100 # 置顶优先级，数字越大越靠前
---

文章摘要

在文章中使用 <!-- more --> 标记来分隔摘要和正文：

这是文章摘要部分。

<!-- more -->

这是文章正文部分。

标签插件

ShokaX 提供了丰富的标签插件来增强文章的表现力：

提示块

成功提示：这是一个成功提示

警告提示：这是一个警告提示

危险提示：这是一个危险提示

信息提示：这是一个信息提示

标签组

标签1
标签1内容

标签2
标签2内容

代码块

console.log('Hello World!');

🔍 SEO 优化

ShokaX 主题内置了良好的 SEO 优化，但你仍需要注意以下几点：

  1. 每篇文章都添加合适的 description 和 keywords
  2. 使用有意义的文章标题和 URL
  3. 添加网站地图

📱 移动端适配

ShokaX 主题已经做了良好的移动端适配，无需额外设置即可在各种设备上正常显示。

🛠 进阶定制

如果想要进一步定制 ShokaX 主题，可以通过以下方式：

注入API

ShokaX 提供了强大的注入 API，可以在不修改主题源码的情况下自定义主题：

# 在 _config.shokax.yml 中添加
injects:
  styles:
    - /css/custom.css
  scripts:
    - /js/custom.js

自定义样式

创建 source/css/custom.css 文件，添加自定义样式：

/* 自定义样式示例 */
.post-title {
  color: #ff6b6b;
}

🆘 常见问题

Q: 为什么我的文章不显示目录？

A: 确保文章中有足够的标题，且在主题配置中启用了目录功能：

post:
  toc:
    enable: true
    number: true

Q: 如何更换主题色？

A: 在 _config.shokax.yml 中修改 theme.profile.color 和 theme.profile.background 即可更换主题色。
